DIVERSITY

DIVERSITY can be run using the command diversity to find different modes of protein-DNA binding. It is also available as a web based application at http://diversity.ncl.res.in.

Prerequisites

  • Python 2.7+ (Not compatible with Python 3.x)
  • python-numpy
  • python-ctypes
  • python-multiprocessing
  • python-re

Download

DIVERSITY can be downloaded from https://github.com/NarlikarLab/DIVERSITY/archive/v1.0.0.tar.gz. It can be run on any Mac or Linux system.

Installation

In order to download and install DIVERSITY, execute the following steps:
  • wget https://github.com/NarlikarLab/DIVERSITY/archive/v1.0.0.tar.gz
  • tar -xvf v1.0.0.tar.gz
  • cd DIVERSITY-1.0.0/NPLB/
  • make
